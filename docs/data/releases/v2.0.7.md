### Main Improvements
This release contains a critical fix for the Hyperbridge cross-chain bridge system. Hyperbridge is a service that connects Polkadot to other blockchain networks, allowing users to move assets and data between different blockchains.

### What Happened
Hyperbridge recently changed from being a full "parachain" (a permanent blockchain connected to Polkadot) to a "parathread" (a temporary, on-demand connection). This change broke the bridge's ability to verify transactions because the system was no longer automatically including Hyperbridge's data in security proofs.
